# Amazon UI Clone

This is a **frontend clone of Amazon's user interface** built using HTML, CSS, and JavaScript. The project replicates the basic layout, design, and UI components of Amazon's homepage for learning and practice purposes.

## Features

- Responsive header with logo, search bar, and navigation menu
- Product listing cards with images, title, price, and ratings
- Sidebar menu for categories (if included)
- Footer with links and information
- Fully responsive design for desktop and mobile screens

## Tech Stack

- **HTML5**
- **CSS3**
- **JavaScript**
- (Optional: Any CSS framework like Bootstrap, Tailwind if used)
